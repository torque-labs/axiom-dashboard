## 4. Gaming Indicator Analysis

### What is Wash Trading?

**Wash trading** = buying and selling the same token within a very short time window (60 seconds) to artificially inflate volume/activity without taking real market risk.

### Flag Definitions

| Flag | Threshold | Description |
|------|-----------|-------------|
| **Rapid Reversals (Critical)** | > 500 | Buy→Sell of same token within 60 seconds |
| **Rapid Reversals (Warning)** | > 100 | Elevated but not critical wash trading |
| **Token Concentration (Critical)** | > 80% | Over 80% of volume in single token |
| **Token Concentration (Warning)** | > 50% | Over 50% of volume in single token |

## 7. Mitigation Strategies

### Immediate Actions

1. **Cooldown Period**: Add 5-minute minimum between buy/sell of same token
2. **Real-time Flagging**: Auto-flag wallets exceeding 100 reversals/day for review

### Future Considerations

1. **Token Diversity**: Require minimum 5-10 tokens for leaderboard eligibility
2. **Multi-Day Participation**: Require 2+ active days for top rewards
3. **Concentration Caps**: Cap PNL contribution from single token at 20-30%
4. **Graduated Rewards**: Apply multiplier (e.g., 1.5x) for clean traders

### Anti-Wash SQL Implementation

```sql
WITH reversal_counts AS (
  SELECT t1."feePayer", COUNT(*) as reversal_count
  FROM axiomtrade_partitioned t1
  JOIN axiomtrade_partitioned t2
    ON t1."feePayer" = t2."feePayer"
    AND t1."tokenOut" = t2."tokenIn"
    AND t2."createdAt" > t1."createdAt"
    AND t2."createdAt" <= t1."createdAt" + INTERVAL '60 seconds'
  WHERE t1."programId" IN (
    'CPMMoo8L3F4NbTegBCKVNunggL7H1ZpdTHKxQB5qKP1C',
    'LanMV9sAd7wArD4vJFi2qDdfnVhFxYSUg6eADduJ3uj'
  )

    'CPMMoo8L3F4NbTegBCKVNunggL7H1ZpdTHKxQB5qKP1C',
    'LanMV9sAd7wArD4vJFi2qDdfnVhFxYSUg6eADduJ3uj'
  )
  AND t1."createdAt" >= '2026-01-19'
```

### Volume Calculation
```
USD Volume = SUM(
  IF tokenIn = USD1 THEN uiAmountIn
  ELSE IF tokenOut = USD1 THEN uiAmountOut
)
```

### Rapid Reversal Detection
```
Count of (buy → sell) pairs where:
  - Same wallet
  - Same token
  - Sell occurs within 60 seconds of buy
```
